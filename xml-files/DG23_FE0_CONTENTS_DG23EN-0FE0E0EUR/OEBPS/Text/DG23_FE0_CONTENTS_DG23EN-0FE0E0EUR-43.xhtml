UPDATING NISSANCONNECT SOFTWARE

[INFOTAINMENT & MULTIMEDIA, #8A78B6, #DBD5EC]

Software updates by default take place automatically. When an update is available you will be notified via a message in the display. To manually check for an update follow the process below.

Navigation System

1

From the Home Menu, Select [All Settings].

2

Touch [System Update].

3

Touch [Software Update].

4

Touch [Start Software Update].

Note:

To automatically be notified of available updates, ensure the update method is set to [Auto].
