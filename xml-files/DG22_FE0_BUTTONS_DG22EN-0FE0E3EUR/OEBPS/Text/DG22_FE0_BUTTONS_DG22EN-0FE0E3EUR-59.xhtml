WINDOW SWITCH

[WINDOW UP DOWN SWITCH]

To open or close the window, push down or pull up the switch and hold it.

AUTOMATIC FUNCTION

The automatic function enables a window to fully open or close without holding the switch down or up.

To fully open the window, push the power window switch down to the second detent and release the switch. To fully close the window, pull the power window switch up to the second detent and release the switch. The switch does not have to be held during window operation.

To stop the window open/close operation during the automatic function, push down or pull up the switch in opposite direction.
