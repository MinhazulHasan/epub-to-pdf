DRIVE MODE SELECTOR

[DRIVE MODE CONTROL, SPORT, ECO, DRIVE MODE SELECTOR]

Touch the up and down arrows on the Drive Mode Selector to select the preferred driving mode:

  • SPORT - Provides higher motor torque and stronger regenerative braking. The setting of the steering system is adjusted to moderately increase steering wheel effort for a sporty feel.
  • STANDARD - This mode will be selected first each time the power switch is placed in the ON position.
  • ECO - Improves energy efficiency/cruising range. Coasting is available in this mode. The mode list will appear in the vehicle information display with the currently selected mode highlighted.
  • [SNOW] MODE (4WD models) - This mode makes it easier to start and drive on snowy roads and frozen roads.
