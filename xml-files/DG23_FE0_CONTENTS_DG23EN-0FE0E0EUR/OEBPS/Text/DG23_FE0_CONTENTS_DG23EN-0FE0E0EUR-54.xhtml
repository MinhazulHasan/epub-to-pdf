PARKING SENSORS

[DRIVING AIDS, #1CACE4, #BCE4FA]

When activated the parking sensor (sonar) system will sound a tone and show in the vehicle information display upon detection of an obstacle or when selecting reverse gear.

1

Obstacle close

2

Nothing detected

3

Obstacle detected

TIP:

Press OK to temporarily deactivate the parking sensor system. Deactivating the Sonar system will also deactivate the REB.

Use the vehicle information display to turn ON/OFF the parking sensor system.

[Settings] -> [Driver Assistance] -> [Parking Aids] -> [Front] & [Rear]

The parking sensor system will be set to ON by default, the state is remembered when the vehicle is restarted.
