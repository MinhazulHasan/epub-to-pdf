Rear Automatic Braking (RAB) system OFF warning light

[BRAKES, RAB, DRIVING AIDS, OFF]

Illuminates when the power switch is placed in the ON position, turns off when the power switch is placed in the READY to drive position.

Illuminates when the RAB system is turned off using the vehicle information display.

If illuminated when the RAB system is on, it may indicate that the system is unavailable.
