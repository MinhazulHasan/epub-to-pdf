EMERGENCY LANE KEEP ASSIST

[DRIVING AIDS, #1CACE4, #BCE4FA]

The Emergency Lane Assist (ELA) system can help to assist you when driving at speeds over 60 km/h (37 MPH) to return the vehicle to the carriageway by applying the brakes to the left or right wheels individually (for a short period of time).

Note:

To change lanes with the ELA system activated, turn on the turn indicator.

The ELA system will be disabled for solid lines for that direction.

When the system intervenes a sudden deceleration can be felt.

To turn the ELA system OFF;

Press the shortcut button on the steering wheel and select [Emergency Lane].

The ELA system will be set to ON by default and will be re-activated each time the vehicle restarts.

System sensitivity

You can set ELA sensitivity using the [Settings] menu in the vehicle information display.

[Settings] -> [Driver Assistance] -> [Lane] -> [Lane Sensitivity]

The state is remembered when the vehicle is restarted.
