RES+ SWITCH

[CRUISE, CRUISE CONTROL, CRUISE CONTROL SWITCHES, CRUISE CONTROLS, CRUISE CONTROL BUTTONS, STEERING WHEEL CONTROLS]

With ProPILOT/Intelligent Cruise Control/Cruise Control/Speed Limiter standby:

Push and release the RES+ switch to resume the last preset speed.

With ProPILOT/Intelligent Cruise Control/Cruise Control/Speed Limiter active:

Push and release the RES+ switch to increase the set speed.
