Intelligent Rear View Mirror has two modes: conventional mode and intelligent mode. Pull the select mode lever to switch to the intelligent Rear View Mirror mode, or push the mode select lever to switch ton conventional mode.

When the intelligent Rear View Mirror mode is on, you choose display settings by pushing the menu button.
